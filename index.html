Update 2021-08-14: the TPL Search Dashboard is still around, but no longer appears to use a WebSocket (at least, a public one that I can find); so this project is currently nonfunctional.

This page uses the same WebSocket feed of anonymized realtime search data from Toronto Public Library as the Search Dashboard. However, it streams it to the Speech Synthesis API of the browser rather than to the screen. Hearing sound depends on your browser's support for speech synthesis.

Not an official TPL thing, made by Alan Harnum using the Infusion Javascript framework. Source code available here.

Use the controls to start and stop the library voices.
