Library Voices

  1. Libraries--Speech synthesis.
  2. Online bibliographic searching--Speech synthesis.
  3. Information retrieval--Speech synthesis.

This page uses the same WebSocket feed of anonymized realtime search data from Toronto Public Library as the Search Dashboard.

However, it streams it to the Speech Synthesis API of the browser rather than to the screen. Hearing sound depends on your browser's support for speech synthesis.

Not an official TPL thing, made by Alan Harnum using the Infusion Javascript framework.
